# Templates Agent Instructions

These files are reusable templates only.

- Do not record live requirements, verification plans, or evidence in `.documentation/templates/`.
- Put workflow rules, ID schemes, status definitions, and agent procedures in `.documentation/AGENTS.md` or `.agents/skills/requirements-traceability/SKILL.md`.
- Use `.documentation/templates/task-template.md` as the shared template for planning tasks under `.documentation/planning/tasks/`.
- Keep templates focused on reusable document shape. Put process authority in the applicable `AGENTS.md` or skill unless field-level placeholder guidance is needed inside the template.
