# Security Policy

## Supported Versions

This project is currently in alpha. Security fixes are applied to the latest released version.

## Reporting a Vulnerability

Please do not report security vulnerabilities in public issues.

Use GitHub private vulnerability reporting if it is enabled for this repository. If it is not available, contact the maintainers privately through the project owner or organization contact channels.

Include:

- A description of the vulnerability
- Steps to reproduce
- Affected versions or commits
- Any known mitigations

## Hardware Safety

This package can communicate with hardware devices. Security or safety-sensitive findings involving device control, serial transport, firmware protocol behavior, or unintended output activation should be reported privately.
