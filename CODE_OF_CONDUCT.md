# Code of Conduct

## Our Pledge

We pledge to make participation in this project a respectful and constructive experience for everyone.

## Expected Behavior

- Be respectful and professional.
- Assume good intent.
- Focus feedback on technical merit.
- Be clear when discussing hardware safety, protocol behavior, and test results.

## Unacceptable Behavior

- Harassment, intimidation, or discriminatory language.
- Personal attacks.
- Publishing private information without permission.
- Disruptive behavior that prevents productive collaboration.

## Enforcement

Project maintainers may remove comments, close issues, reject contributions, or restrict participation when behavior violates this code of conduct.

To report a concern, contact the maintainers through the repository's GitHub issue or security channels.
