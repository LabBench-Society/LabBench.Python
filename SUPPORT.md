# Support

## Getting Help

For bugs, feature requests, and protocol questions, use GitHub Issues:

https://github.com/LabBench-Society/LabBench.Python/issues

When asking for help, include:

- Package version
- Python version
- Operating system
- Device type and firmware version, if relevant
- Serial port and connection details, if relevant
- Minimal code or command that reproduces the issue

## Hardware Questions

For hardware problems, include the device model, connected accessories, expected behavior, observed behavior, and whether the issue is reproducible with the provided examples.
